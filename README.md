# Hesap Makinesi (Python)
Python ile yazılmış tekrarlayan Hesap Makinesi

Kullanılan fonksiyonlar :
* if / elif / else
* While
* print()
* input()
* isinstance()
* float(input())
* int(input())

Yapılabilecek işlemler :
* [1] => Toplama (Tekrarlı)
* [2] => Çıkartma (Tekrarlı)
* [3] => Çarpma (Tekrarlı)
* [4] => Bölme (Tekrarlı)
* [5] => Kalan alma (Tekrarlı)
* [99] => Çıkış
